## 🎨 Customization Tips

### Change Animation Speed
Edit durations in `app/globals.css`:
```css
animation: fadeInUp 0.8s ease-out forwards; /* Change 0.8s */
```

### Change Colors
Edit CSS variables in `app/globals.css`:
```css
--primary: #9B5DE5; /* Change to your color */
```

### Add More Animations
Use the same pattern in globals.css:
```css
@keyframes yourAnimation {
  from { /* your styles */ }
  to { /* your styles */ }
}
```
