## 🎨 Customization Guide

### Change Primary Color
Edit `app/globals.css`:
```css
:root {
  --primary: #9B5DE5;  /* Change this purple to your color */
}
```

### Adjust Animation Speeds
Edit keyframe durations in `app/globals.css`:
```css
@keyframes fadeInUp {
  /* Change 0.8s to your preferred duration */
  animation: fadeInUp 0.8s ease-out forwards;
}
```

### Mobile Animation Timing
Adjust in `@media (max-width: 768px)` section in `app/globals.css`
